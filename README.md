# NPPSC Dynamics
### Nonlinear Phase-Space State Classification

This framework studies coupled nonlinear discrete dynamical systems, specifically focusing on the intersection of **Logistic-Map Lattices** and **Phase-Space Geometry**.

## The Four Pillars
- **Dynamics:** 3D coupled nonlinear feedback loops.
- **Stability:** Lyapunov exponent tracking for sensitivity analysis.
- **Measures:** Invariant probability density estimation.
- **Geometry:** Separatrix manifold extraction for basin boundary definition.
